`/home/tehlappy/Desktop/AI/abyssal-assets/cp2077_mods/tweakdb/custom_models_msn_abyssal.yaml`

The pack is organized around:

- Abyssal/Nessie guardian models

- Freighter/fighter vehicles
- Loch Exchange and communion props

Asset guardrail: the RTX 3060 profile is the default performance budget. Prefer LODs, trim sheets, material variants, and streamed hero models over heavy unique textures.
